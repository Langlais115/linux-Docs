==
at
==
**at** command is use to **one time schedule a command**
    **Ex:**

    at mysqldump





Create file
.. code-block:: bash

    `touch [fileName]`



copy

$ echo $null >> [fileName]

Delete file
    $ rm [fileName]

Display file content

    $ cat > [filename]

    $ echo $null >> [filename]


Edit text file

    $ vi [fileName]

    $ nano [fileName]

    $ emacs [fileName]

    $ pico [fileName]

    $ ed [fileName]

Copy file or folder
    $ cp [fileName] [newFileName]

    $ cat [fileName] > [newFileName]

    $ dd if=[fileName] of=[newFileName]

Move/Rename file or folder
$ mv [fileName] [newFileName]
